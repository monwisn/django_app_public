  4. Contact form (with send to me- sending a copy of the message to the user)

  14. Blog- information about the currently logged in user
  15. Autocomplete author of post (logged-in user)
  16. Account- Profile information of the logged-in user (username in brackets)
  17. Sending an email to subscribers when a new blog post has been added (only if the status is 'Published')
  18. Editing and deleting blog posts by author (only logged-in user, only the author can edit his posts)
  19. Deleting posts by author (or admin-admin site)
  20. Control panel-only for admin or logged-in users with 'staff member' privileges
  21. Control panel-sending push notifications

  23. Delete the user account with all blog posts
